## еҝ«йҖҹејҖе§Ӣ

### зҺҜеўғиҰҒжұӮ

- Java 21+
- Maven 3.8+
- MySQL 8.0
- Redis 7.0
- RabbitMQ 3.x
- Ollama (еҸҜйҖүпјҢз”ЁдәҺ AI иғҪеҠӣ)

### 1. еҗҜеҠЁдёӯй—ҙд»¶

```bash
# MySQL (йңҖйў„е…ҲеҲӣе»әж•°жҚ®еә“)
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS \`mindpulse-database\` DEFAULT CHARSET utf8mb4;"

# Redis
docker run -d -p 6379:6379 --name redis redis:7

# RabbitMQ
docker run -d -p 5672:5672 -p 15672:15672 --name rabbitmq rabbitmq:3-management

# Ollama (AI жңҚеҠЎ)
ollama run qwen2.5:1.5b
```

### 2. еҲқе§ӢеҢ–ж•°жҚ®еә“

```bash
mysql -u root -p mindpulse-database < sql/init-tasks.sql
mysql -u root -p mindpulse-database < sql/init-notes.sql
mysql -u root -p mindpulse-database < sql/init-reminders.sql
```

### 3. й…ҚзҪ®зҺҜеўғеҸҳйҮҸпјҲеҸҜйҖүпјү

```bash
export JWT_SECRET="your-secret-key-at-least-32-chars"
export AI_SERVICE_URL="http://localhost:8000/api/agent"
export RABBITMQ_HOST="localhost"
export REDIS_HOST="localhost"
```

### 4. еҗҜеҠЁжңҚеҠЎ

```bash
# ејҖеҸ‘жЁЎејҸ
mvn spring-boot:run

# жү“еҢ…иҝҗиЎҢ
mvn clean package -DskipTests
java -jar target/mindpulse-system-0.0.1-SNAPSHOT.jar
```

жңҚеҠЎеҗҜеҠЁеҗҺпјҡ

- API ж–ҮжЎЈпјҡhttp://localhost:8090/doc.html
- OpenAPI JSONпјҡhttp://localhost:8090/v3/api-docs
- RabbitMQ з®ЎзҗҶеҗҺеҸ°пјҡhttp://localhost:15672 (guest/guest)
